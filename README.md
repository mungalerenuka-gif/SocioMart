# SocioMart

## Buy, Sell & Donate Within Trusted Communities

SocioMart is a web-based community marketplace designed for residential societies where users can securely buy, sell, and donate unused items within their own society, nearby communities, or public listings.

## Problem Statement

Many residential societies currently exchange unused items through WhatsApp groups, verbal communication, or society meetings, which often leads to lost information, lack of organization, limited visibility, and difficulty in connecting buyers, sellers, and donors. SocioMart provides a centralized digital platform to solve this problem.

## Objectives

* Create a trusted local marketplace
* Simplify buying and selling unused household items
* Support donation to organizations
* Reduce waste through reuse
* Enable secure communication between users

## Features

* User Registration and Login
* Society Token Verification
* Sell Items with Category Selection
* Image Upload for Products
* Buy Items with Search and Filters
* Private Chat Between Buyer and Seller
* Donation Zone with Organization Selection
* Visibility Control (Society / Nearby Society / Public)
